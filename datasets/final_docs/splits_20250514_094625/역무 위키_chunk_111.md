## 본문 내용

정기승차권 현금영수증 안내

발급절차

국세청 홈텍스(www.hometax.go.kr) → 조회/발급

현금영수증 발급수단 → 소비자 발급 수단 관리

카드번호 입력란에 카드번호 13자리 등록(10자리인 경우 앞 111 추가) 내역확인 : 로그인⇒ 조회/발급 ⇒ 현금영수증조회⇒ 사용내역조회 ⇒ 월별내역조회

소득공제 : 교통카드(마이비, 하나로 카드 등)

각 카드사 홈페이지 회원 등록

등록 시점 이후부터 공제 가능

16강 휴대금지 제한품
